## Setup

Esta landing page es un front-end estático y no requiere una configuración de backend compleja. Para desplegar y visualizar este proyecto, sigue estos pasos:

1.  **Clonar el repositorio:**
    ```bash
    git clone https://github.com/tu-usuario/orbit-landing-page.git
    cd orbit-landing-page
    

2.  **Instalar dependencias (si aplica, para un proyecto React/Next.js):**
    Si la landing page se construye con un framework como Next.js o React, necesitarás instalar las dependencias. Para un HTML/CSS/JS plano, omite este paso.
    ```bash
    # Ejemplo para un proyecto Next.js
    npm install
    # o
    yarn install
    

3.  **Ejecutar el proyecto localmente (si aplica):**
    Para proyectos basados en frameworks.
    ```bash
    # Ejemplo para un proyecto Next.js
    npm run dev
    # o
    yarn dev
    
    Tu aplicación estará disponible en `http://localhost:3000` (o el puerto configurado).

4.  **Para un despliegue estático (HTML/CSS/JS puro):**
    Simplemente abre el archivo `index.html` en tu navegador web. Para producción, puedes subir los archivos a cualquier servidor estático (Netlify, Vercel, GitHub Pages, etc.).
